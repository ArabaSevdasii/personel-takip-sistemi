# 🏢 Personel ve Maaş Takip Sistemi

Babam için yapay zeka ile geliştirdiğim, kullanımı son derece kolay, modern arayüzlü personel ve maaş takip otomasyonu.
Şifre:1234

## 🚀 Özellikler
* **Günlük Devamsızlık Takibi:** Pazartesi - Cuma arası saatlik, yarım günlük veya tam günlük kesinti hesaplama.
* **Otomatik Net Maaş:** Taban maaş üzerinden yapılan kesintileri anında hesaplayıp net ödenecek tutarı gösterme.
* **Geçmiş Ödemeler Arşivi:** Haftalık maaşlar ödendiğinde verileri silmez, tarih ve saat damgasıyla arşive kaldırır.
* **Soft Delete (Güvenli Silme):** İşten ayrılan personel listeden kaldırılır ancak geçmiş muhasebe kayıtları korunur.
* **Kullanıcı Dostu Arayüz:** Yaşlı veya teknolojiye uzak esnaf kullanıcılar için devasa butonlar ve renk kodlamalı (Kırmızı/Yeşil) tasarım.

## 💻 Kullanılan Teknolojiler
* **Dil:** Python 3
* **Arayüz (GUI):** CustomTkinter
* **Veritabanı:** SQLite (Veriler yerel olarak `.db` dosyasında güvenle saklanır)
